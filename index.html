Программы для 3D моделирования

Предназначены программы для 3D модельеров — они создают 3D модели персонажей, зданий, техники, животных. Для визуализация интерьеров помещений, экстерьеров, выставочных стендов. Для создания наружной рекламы, печатной продукции, а так же дизайна сайтов. Программы для обработки изображений

Предназначены программы для того, чтобы улучшить качество картинки, настроить параметры цвета или применить к снимку фильтр. Для обрезания, отражения, выравнивания и изменения размера кадра, для удаления объектов и замены фона. Также с этими программами можно делать фотомонтаж, коллажи, макеты, ретушь портретов, рисовать, составлять инфографики и комиксы, делать анимацию, конвертировать форматы, оптимизировать изображения.

Программы для редактирования видео и аудио

Предназначены программы для видео для того, чтобы осуществлять монтаж видео и звуковых файлов. Кроме того, большинство видеоредакторов позволяют создавать и накладывать титры, осуществлять цветовую и тональную коррекцию изображения, микшировать звук и создавать спецэффекты.Программы для аудио предназначены для удаления шума, запись с одновременным прослушиванием имеющихся дорожек, запись с микрофона.
